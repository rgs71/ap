## CS50 Sandbox

Temporäre Programmierumgebung für Schüler und Lehrer <https://sandbox.cs50.io/>

## Kommandozeilen Programme

#### `help50`

Gibt weniger komplex formulierte Antworten für die Programmiersprache `C` beim verwenden der Compilers oder von Valgrind.
Verwendung: `help50 ./program command line args` or  
`help50 valgrind ./program command line args`
 
#### `debug50`

Führt ein Programm Schritt für Schritt, Zeile für Zeile, aus, um zu sehen was passiert
Verwendung: `debug50 ./program command line args` bzw. `debug50 python programm.py`

#### `style50`

Kotrolliert den Stil eines `C`Programms in Bezug auf [CS50 style guide](https://cs50.readthedocs.io/style/c/)  
Verwendung: `style50 program.c`

#### `check50`

Kontrolliert ein Programm bezüglich vorgegebener Testfälle
Verwendung: `check50 cs50/problems/2018/ap/problem_name`
